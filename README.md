# Корпоративный лендинг

**Демо:** [ссылка на Vercel/Netlify]  
**Стек:** HTML5, CSS3, JavaScript (ES6)

## Описание
Адаптивный лендинг для компании/услуги. Реализован:
- Адаптивная верстка (mobile-first)
- Бургер-меню для мобильных
- Аккордеон (разворачивающиеся блоки с вопросами)
- Плавный скролл к секциям

## Локальный запуск
\`\`\`
git clone ...
Открыть index.html
\`\`\`

## Структура
- `index.html` – разметка
- `styles.css` – стили
- `script.js` – логика (бургер, аккордеон)
- `img/` – изображения
